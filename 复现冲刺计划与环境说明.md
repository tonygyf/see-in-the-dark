## 5. 4周冲刺版（复现+学习并行）
### 第1周：论文拆解 + 基线脚手架
- 目标：把论文转成可编码任务，跑通最小训练流程。
- 复现任务：
  - 拆解论文模块：`SCM / DSF / TSR / Loss`
  - 设计代码结构：`src/datasets`、`src/models`、`src/train.py`、`src/eval.py`
  - 跑通一个最小样例（随机数据也可）验证训练循环。
- 学习任务（每天）：
  - Python：函数、类、文件组织、异常处理
  - PyTorch：`Tensor`、`Dataset/DataLoader`、`forward/backward`
- 周交付：
  - 一份模块流程图
  - 一版可运行训练脚手架

### 第2周：复现基线检测器
- 目标：先得到不含 `SCM/DSF/TSR` 的基线结果。
- 复现任务：
  - 搭建主干网络和检测头
  - 实现基础损失（如分割/回归项）
  - 打通评估流程与可视化输出
- 学习任务（每天）：
  - 深度学习：卷积、特征图、损失函数、优化器
  - PyTorch：`nn.Module`、参数更新、学习率调度
- 周交付：
  - 基线模型可训练
  - 首轮指标与失败案例截图

### 第3周：加入SCM与空间约束
- 目标：实现论文关键改进之一 `SCM + Lsr/Lss`。
- 复现任务：
  - 在训练阶段接入辅助模块SCM
  - 实现 `Lsr`（重建约束）与 `Lss`（语义约束）
  - 对比实验：`Baseline` vs `Baseline+SCM`
- 学习任务（每天）：
  - 多分支网络与辅助监督
  - 特征对齐、上采样、mask监督
- 周交付：
  - 消融表（至少一组）
  - 中间特征可视化（文本区域热力图）

### 第4周：加入DSF+TSR并完成报告
- 目标：拼齐完整方法并整理复现结论。
- 复现任务：
  - 加入 `DSF`（动态蛇形特征建模）
  - 加入 `TSR`（旋转矩形累积塑形）
  - 完成总损失：`Lt = Lseg + LH + Lθ + Lss + Lsr`
  - 输出最终指标、案例分析、与论文差异说明
- 学习任务（每天）：
  - 注意力机制、几何参数回归 `(x, y, h, w, θ)`
  - 后处理与形态学操作思想
- 周交付：
  - 最终复现报告（含指标/可视化/误差分析）
